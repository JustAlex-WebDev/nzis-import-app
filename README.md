# НЗИС Import Application  

## Overview  
The НЗИС Import Application is a test project designed to streamline the process of importing patient data from the НЗИС (National Health Information System). This application allows users to input a patient's PID or HPH and retrieve their associated referrals and related details.  

## Features  
- **Patient Data Import**: Input a patient's PID or HPH to fetch their data from the НЗИС system.  
- **Referral Management**: View all referrals associated with the patient.
